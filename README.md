# EvtxTool
[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

 EvtxTool, Tool to Upload Evtx logs to ELK

### USAGE

```sh
$ git clone https://github.com/MazX0p/EvtxTool.git
$ cd EvtxTool
$ python3 EvtxTool.py
```
